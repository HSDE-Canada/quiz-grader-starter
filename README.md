# Week 5 - Quiz Grader

## Project Goal
Build a Quiz Grader where the frontend sends quiz answers to a backend server, and the server returns the score and feedback.

This project focuses on frontend/backend communication.

## Instructions
Work in the `starter` folder of this repository.

You will work in both the frontend and server code:

- `starter/frontend/script.js`
- `starter/server/server.js`

Before running the server, open the `starter/server` folder in the terminal and run:

```bash
npm install
node server.js
```

## Challenges

1. Personalize the quiz questions and answers.
2. Update the server’s correct answers.
3. Send the selected answers to the server.
4. Write the server request handler.
5. Convert the server response to JSON.
6. Display the percentage.
7. Display the score and total.
8. Display the feedback message.
9. Update the circular score graphic.

## Extension Challenges
After the required app works, you may improve the quiz wording, answer choices, feedback messages, and visual theme.
